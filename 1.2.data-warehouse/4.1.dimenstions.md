### **ğŸ“Œ 2. Slowly Changing Dimensions (SCD)**

| -------------- | ----------------------------------------------- |

| **SCD Type 2** | Maintains historical changes using versioning.  |
| **SCD Type 3** | Stores both old and new values in the same row. |
